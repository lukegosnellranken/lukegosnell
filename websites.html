St. Charles Veterans Museum

Code | Contact for Live Demo

After taking a 27-hour online course over building custom themes for WordPress, I voluntarily built a new website for the St. Charles County Veterans Museum. Using my knowledge of the WordPress process, along with writing custom PHP, I created a brand new WordPress theme, as opposed to using a custom theme like I had done for my previous projects. I designed both the front and back ends of this site, utilizing my own custom post types so that I could showcase to the museum owner how to easily create, edit, or delete news, events, and veteran stories. The JSON data from these posts are then displayed and archived automatically through PHP. Additionally, I utilized both JavaScript and PHP to create a “Veterans Search,” which can be used anywhere on the site. I also designed an Android app, which would have worked in conjunction with the website, pulling information about veterans and their stories from a database via scanning QR codes.
